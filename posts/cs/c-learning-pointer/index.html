指针是 C 语言中一个重要的概念，摘取几个重要的点复习下。

是什么#

指针，英文名 pointer，缩写为 ptr，本质是一个表示内存地址的值，类似于 0x7fffffffd904 这种。

* 运算符 和 & 运算符#

  • * 运算符跟在类型关键字的后面用来进行指针的声明，以下三种写法都是正确的
    int* a;
int * a;
int *a;
    但是同一行声明两个指针时，需遵循如下写法
    int *a,  *b //正确写法
int* a, b //这种写法错误
  • * 运算符放在变量的前面用于取出存在对应内存地址的值，或者进行赋值
    *p = 12
  • & 运算符用来取出对应变量的内存地址

指针变量的初始化#

正确写法：

int* a;
int b;
a = &b;
*a = 12;

错误写法：

int* a;
*a = 12;

原因：指针变量在被声明后，编译器为指针变量分配了一个内存空间，但其在内存空间中对应的内存地址是随机的，此时若是读写指针变量，读写的是一个随机地址的值，并不能获取预期的结果。

指针间的运算#

  1. 指针间可以进行减运算，将返回两个指针之间的距离，为 ptrdiff_d 类型。
  2. 指针间不能进行加法运算，但是可以加减对应的值，来表示其他地址。
  3. 指针不能进行乘除运算。
  4. 指针间可以进行比较运算，用于比较谁的地址更大，返回 0 或者 1。
